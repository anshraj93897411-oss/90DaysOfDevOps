                                Task-File System Hierarchy
                                  
Linux File System Hierarchy-File system Hierarchy in Linux is a strutured upside-down tree of folders.It starts with a single base called Root(written as /) and every thing in computer like programs,user files ,hardware devices are branches out from this root directory. 

/ (root) - The starting point of everything
/home - User home directories
/root - Root user's home directory
/etc - It is a directory which is a configuration center.It contains text file which tell your operating system (OS) and installed programe that how to behave,start up or connect to networks
/var/log -/var/log is directory in Linux that stores all system,service,applications and kernel log files.
/tmp - /tmp is directory where temporary files are stored here user can create and modify files. Here generally data is deleted automatically upon restarting the system.
