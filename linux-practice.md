■ Check running processes
In Linux,Check running processes is a process of looking live list of applications,task or background services currently active on your computer.
commands used for checking running processes are ps,top,htop and etc.
ps -Displays static snapshot of running processes
top -Displays realtimme and dynamic CPU and memory usage
htop -Displays user friendly,color coded and graphical view of processes

■ Inspect one systemd service.
Inspect systemd service in Linux means to check current status,details and logs to see if it is running correctly.
command used are systemctl status,systemctl list-units and etc
