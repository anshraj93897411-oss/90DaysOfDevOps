■ Process management
ps -Displays static snapshot of running processes.
top -Displays realtime and dynamic CPU and memory usage.
htop -Displays user friendly,color coded and graphical view of processes.
pstree -Displays processes in tree structure to see parent child links.
pidof - it returns process ID(PID) of a specific running program.
■ File system
pwd -It tell about present working directory.
ls -It list all file under directory.
ls -a -It list all file Under directory including hidden files.
ls -l -for ditailed attributes.
tree -Displays a visual ,tree structure of directories.
cat -outputs the entire raw content of file to terminal panel.
less-open file in scroblle view,ideal for reading large logs and scripts.
head/tail -displays first or last 10 lines of file respectively

■ Networking troubleshooting
ip addr -check if internet interfaces are UP and have valid assigned IP addresses
ping -It test  basic network connectivity and packet loss to a specific destination
dig -verifies if DNS servers are properly translating domain name into IP addresses
