■ Creating a file
touch: It creates new file.
 
■ Writing text to a file
 vi/vim : It is use to write text in a file.
 
■ Appending new lines
Append(>>) : It add new text to a file without deleting current (existing) data.
Redirect(>) : It add new text to a file and it delete current (existing) data.

■ Reading the file back
cat: It reads file.
head : It reads first 10 lines of a file by default.
tail :It reads last 10 lines of a file by default.
